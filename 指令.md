git checkout [文件名] #恢复某个文件的本地修改到最新的提交上 // 比如乱改了一个文件恢复到 拉取时候的状态
git checkout . // 恢复所有文件
git checkout [commit ID] [file] #恢复某个commit的指定文件到暂存区和工作区 【commit】 就是 某次提交的版本号 
/ 这个意思就是恢复到某次提交的 文件  与 第一个注意区分 第一个是恢复到最新的 
git reset [文件名] // 重置暂存区的指定文件 与上次 commit 一样 但是工作区不变
git reset --hard // 重置暂存区与工作区，与上一次commit保持一致
